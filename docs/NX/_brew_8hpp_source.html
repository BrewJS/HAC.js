15  ----- Brew.js - C++ userland JavaScript API for Nintendo homebrew -----

18 
19  Using duktape engine (https://duktape.org), base of the JS API.
20 
21  FAQ:
22 
23  - Which ECMAScript version if supported?
24  According to duktape's documentation, duktape's JavaScript runs like ES5 (although some API of ES5 is not supported).
25  It also has some ES6 features. read more about this on duktape's GitHub: https://github.com/svaarala/duktape/blob/master/README.md
26 
27  - Which Nintendo OSs are supported?
28  Currently Brew.js runs under Nintendo Switch, Nintendo 3DS and Nintendo DS. More platforms are planned, like Wii or GBA,
29  but we plan to fully support these 3 current OSs before starting with a new one.
30 
31  - Do I have to add something extra to embed Brew.js on my project?
32  Theoretically no. Brew.js automatically should detect the current OS we're compiling for, and it'll error if it fails detecting it.
33  Everything should be automatically done.
34 
35  - Where can I get more info or docs?
36  You have the documentations and the repository here: https://XorTroll.github.io/Brew.js
