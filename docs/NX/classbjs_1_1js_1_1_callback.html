Detailed Description

Class representing a callback, which is a function as an object. To be used inside API JS functions/methods.

Constructor & Destructor Documentation

◆ Callback()

bjs::js::Callback::Callback ( NativeContext  Context,
u32  Index 
)

Gets the callback function from the given index using the given context. If the argument is not a function throws an error.

Parameters
ContextThe context to use it with.
IndexThe index of the argument of the API JS function.
Note
This should NOT be CALLED manually. use FunctionHandler::GetCallback(...) to get it.
